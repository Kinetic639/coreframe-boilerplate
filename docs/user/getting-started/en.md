# Getting Started with AmbraWMS

Welcome to AmbraWMS - your comprehensive warehouse management solution. This guide will help you get started with the system and understand its core features.

## What is AmbraWMS?

AmbraWMS is an enterprise-grade warehouse management system built with modern technologies:

- **Next.js 15** - Fast, modern web framework
- **Supabase** - Secure database and authentication
- **Multi-tenancy** - Organization and branch support
- **Polish Compliance** - Built for Polish warehouse regulations

## Key Concepts

### Organization Structure

AmbraWMS uses a hierarchical structure:

1. **Organization** - Your company
2. **Branch** - Each warehouse (branch = warehouse)
3. **Location** - Storage bins/shelves within a warehouse

### Products and Inventory

- **Products** - Base product information
- **Variants** - Different sizes, colors, or forms
- **Inventory Data** - Stock levels per warehouse
- **Stock Movements** - Track all inventory changes

## Quick Start Checklist

Follow these steps to get your warehouse up and running:

- [ ] Set up your organization profile
- [ ] Create warehouse branches
- [ ] Add storage locations
- [ ] Import or create products
- [ ] Configure reorder points
- [ ] Add your first stock movement

## Navigation

The main navigation is organized by modules:

- **Home** - Dashboard and announcements
- **Warehouse** - Inventory management
- **Teams** - Collaboration tools
- **Organization** - Admin settings
- **Support** - Help and documentation
