# Pierwsze Kroki z AmbraWMS

Witamy w AmbraWMS - kompleksowym systemie zarządzania magazynem. Ten przewodnik pomoże Ci rozpocząć pracę z systemem i zrozumieć jego podstawowe funkcje.

## Czym jest AmbraWMS?

AmbraWMS to system zarządzania magazynem klasy enterprise zbudowany w oparciu o nowoczesne technologie:

- **Next.js 15** - Szybki, nowoczesny framework webowy
- **Supabase** - Bezpieczna baza danych i uwierzytelnianie
- **Multi-tenancy** - Wsparcie dla organizacji i oddziałów
- **Zgodność z Polskimi Przepisami** - Zbudowany zgodnie z polskimi regulacjami magazynowymi

## Kluczowe Koncepcje

### Struktura Organizacyjna

AmbraWMS używa struktury hierarchicznej:

1. **Organizacja** - Twoja firma
2. **Oddział** - Każdy magazyn (oddział = magazyn)
3. **Lokalizacja** - Regały/półki wewnątrz magazynu

### Produkty i Inwentarz

- **Produkty** - Podstawowe informacje o produktach
- **Warianty** - Różne rozmiary, kolory lub formy
- **Dane Inwentarzowe** - Stany magazynowe per magazyn
- **Ruchy Magazynowe** - Śledzenie wszystkich zmian w inwentarzu
